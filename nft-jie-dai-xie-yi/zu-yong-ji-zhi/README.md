# 租用机制

### 交易市场

Soup Finance NFT 交易平台可以让用户们以免信任的方式列出他们的数字资产以供出租。该平台将支持来自公共协议的原生与包装NFT。

### 包装资产

Soup Finance 协议将支持任意资产的转换和跟踪。它遵循与包装代币相同的机制，相关的原始资产将锁在智能合约上，然后铸造一个包装资产作为临时代品或代币。

包裹资产依然保留原始资产的功能，只有在铸造出的包裹资产租用期限到期并被烧毁时才能解锁。 Soup Finance 铸造包装资产与游戏项目租赁协议将是即插即用的机制，包裹资产将可以漫游去任何Soup Finance支持中的公共区块链。

### 出借方&#x20;

1. 出借方可以列出支持资产的设置条款，例如费用和租赁期限。
2. 链上结算资产报价后，被打包的资产被转移到租户的地址，然后出借方将收到租金。
3. 租用期限结束后，打包的NFT相等于过期品然后进行销毁，而原始NFT可被出借方索回。

### 租借方

1. 租户可以在交易平台上进行租借NFT资产，租户可以选择接受出租方所给出的条件或者可以提供报价（费用和租赁期限）。
2. 在链上结算资产报价后，被打包的资产会转移到租户的地址即可供给租户使用。
3. 租用期限结束后，被打包的资产将进行销毁，而原NFT资产会还回给物主。
